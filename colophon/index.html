Colophon

How this thing is made — the press notes for the shop.

Type

Archivo sets the display — the big, tight, confident headlines. Inter carries the reading text. IBM Plex Mono does the labels, tags, and job-ticket marks — the shop stamps.

All-sans system (Fraunces retired). Loaded with font-display: swap.

Color

A 90s geometric color system: eight themes on one skeleton, 64 colors in all, driven by role tokens on a constant white ground. The client work keeps each brand’s real, pinned colors; everything else is a costume the structure wears.

Stack

Built with Vite + React, animated with Framer Motion, deployed on Netlify. No page router — three static entries and a system of layers.

The shapes

Six shapes — one per project — re-choreograph through a roster of compositions, morphing only by border-radius and rotation. A seeded, constrained-random walk picks the next arrangement; the palette advances with it. Tap a shape and it floods to the project’s real color and opens.
